# 📱 iPhone 快捷指令 · Apple Watch 数据自动同步指南

## 概述

通过 iPhone 自带的「快捷指令」（Shortcuts）App，你可以在每天运动结束后自动将 Apple Watch 的健康数据发送到 GlowFit 服务器，无需手动输入。

---

## 准备工作

1. 确保你的 GlowFit 后端服务已经部署并可公网访问（或局域网内可访问）
2. 记下你的服务器地址，例如：
   - 本地测试：`http://192.168.1.100:3000`
   - 公网部署：`https://your-domain.com`

---

## 🛠 配置快捷指令

### 步骤 1：打开快捷指令 App

在 iPhone 上打开「快捷指令」App（找不到的话在主屏幕下滑搜索 "快捷指令"）

### 步骤 2：创建新快捷指令

1. 点击右上角 **「+」** 号
2. 点击 **「添加操作」**
3. 在搜索框中搜索 **「健康」**，选择 **「获取健康样本」**

### 步骤 3：获取运动热量数据

1. 在「获取健康样本」操作中：
   - 样本类型：**「活动能量」**
   - 开始日期：选择 **「今天」**
   - 结束日期：选择 **「今天」**
2. 再添加一个「获取健康样本」：
   - 样本类型：**「步数」**
   - 时间范围：今天
3. 再添加一个「获取健康样本」：
   - 样本类型：**「步行+跑步距离」**
   - 时间范围：今天

### 步骤 4：计算总量

1. 添加 **「计算统计信息」** 操作（对活动能量使用）：
   - 计算方法：**「求和」**
   - 单位：**「千卡」**
2. 再添加一个「计算统计信息」（对步数使用）：
   - 计算方法：**「求和」**
3. 再添加一个「计算统计信息」（对距离使用）：
   - 计算方法：**「求和」**

### 步骤 5：获取当前日期

1. 添加 **「日期」** 操作：
   - 设置为 **「当前日期」**
   - 日期格式：**「自定义」** → `yyyy-MM-dd`

### 步骤 6：发送 HTTP 请求

1. 添加 **「URL」** 操作：
   - 输入：`你的服务器地址/api/sync-watch-data`
   - 例如：`http://192.168.1.100:3000/api/sync-watch-data`
2. 添加 **「获取内容」** 操作（或叫「下载 URL 内容」）：
   - 方法：**POST**
   - 头部（Headers）：
     - 键：`Content-Type`，值：`application/json`
   - 请求体（Request Body）：
     - 类型：**JSON**
     - 内容：
       ```json
       {
         "activeCalories": "{{活动能量总和}}",
         "steps": "{{步数总和}}",
         "distance": "{{距离总和}}",
         "date": "{{当前日期}}",
         "exerciseMinutes": 0,
         "standHours": 0
       }
       ```
     - 注意：`{{ }}` 表示变量，点击后选择前面步骤的输出

### 步骤 7：显示结果（可选）

1. 添加 **「显示提醒」** 操作：
   - 标题：`GlowFit 同步完成 ✅`
   - 内容：`已同步今日健康数据到 GlowFit`
   - 勾选「运行时显示」

## 🎯 设置自动运行

### 方法一：每天定时运行

1. 在快捷指令列表中找到你刚创建的指令
2. 点击 **「自动化」** 标签（底部 Tab）
3. 点右上角 **「+」** → **「创建个人自动化」**
4. 选择 **「特定时间」**
5. 设置时间为每天晚上（例如 **21:00**），选择「每天」
6. 点击「下一步」
7. 添加操作 → 搜索并选择 **「运行快捷指令」**
8. 选择你刚创建的同步指令
9. 取消勾选「运行前询问」
10. 点击「完成」

### 方法二：结束体能训练时自动运行

1. 创建自动化 → **「体能训练」**
2. 选择 **「停止体能训练时」**
3. 选择你的同步快捷指令
4. 取消「运行前询问」

## 🔒 安全提示

- 如果服务器暴露在公网，建议添加简单的 Token 认证
- 可以在快捷指令的 Header 中添加 `Authorization: Bearer your-token`
- 服务端相应添加 Token 校验（参考 server.js 中的注释）

## 📋 测试验证

配置完成后：

1. 手动运行一次快捷指令
2. 访问 `你的服务器地址/api/watch-data/latest` 查看最新数据
3. 打开 GlowFit 运动页面，查看 Apple Watch 数据是否更新

## 🆘 常见问题

**Q：同步后数据没有更新？**  
A：检查服务器地址是否正确，确保 iPhone 和服务器在同一网络（或服务器已部署到公网）

**Q：步数或距离为 0？**  
A：确保在「获取健康样本」中选择了正确的数据类型和时间范围

**Q：能否同步心率？**  
A：可以！在快捷指令中再添加「获取健康样本」选择「心率」，计算平均值后一并发送
